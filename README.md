# Optimization Techniques in AI and Machine Learning

This repository contains implementations of four popular nature-inspired optimization algorithms: **Whale Optimization Algorithm (WOA)**, **Cuckoo Search Algorithm (CSA)**, **Krill Herd Algorithm (KHA)**, and **Lion Optimization Algorithm (LOA)**. These techniques are widely used for solving complex optimization problems in AI, machine learning, and engineering.

### 3. **Krill Herd Algorithm (KHA)**
- **Inspired By**: The collective foraging and movement of krill swarms.
- **Key Features**:

  - Strong convergence for high-dimensional optimization.
- **Applications**: Power system optimization, neural network training, portfolio optimization.

### 4. **Lion Optimization Algorithm (LOA)**
- **Inspired By**: Social hierarchy and hunting strategies of lion prides.
- **Key Features**:
  - Efficient balance between global and local search.
  - Utilizes nomadic and pride-based search strategies.
- **Applications**: Sensor network optimization, supply chain management, hyperparameter tuning.
